<h2>Description</h2>

<p>
This project presents an interactive ESG (Environmental, Social, Governance) Indicators Dashboard for anonymous company rankings. The dashboard allows users to select a sector (Electrical, Mechanical, etc.) and navigate through environmental, social, and governance indicators for performance analysis. Each KPI is visualized with rankings and ratings, providing actionable insights for sustainability and corporate responsibility evaluation.
</p>

---

<h2>Dashboard Structure</h2>

<ul>
  <li><b>Sector Selection Page:</b> Users select the sector of interest (Electrical, Mechanical, etc.).</li>
  <li><b>Home Page:</b> Choose the ESG pillar: Environmental, Social, or Governance.</li>
  <li><b>Environmental Page:</b> Shows KPIs such as:
    <ul>
      <li>Greenhouse Gas Indicator</li>
      <li>% Non-Hazardous Waste</li>
      <li>% Hazardous Waste</li>
      <li>Rating and Ranking</li>
    </ul>
  </li>
  <li><b>Social Page:</b> Shows KPIs such as:
    <ul>
      <li>Sum of Training Sessions per Number of Employees</li>
      <li>Number of Employees</li>
      <li>Absenteeism Rate</li>
      <li>Rating and Ranking</li>
    </ul>
  </li>
  <li><b>Governance Page:</b> Shows KPIs such as:
    <ul>
      <li>Number of meetings with CSR topics on board agenda</li>
      <li>% Independent Administration</li>
      <li>Rating and Ranking</li>
    </ul>
  </li>
</ul>
